# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-05

Initial release.

### Features
- Visual window counting with pie chart icons (Nerd Font hexagonal icons)
- Custom format strings with placeholders ({icon}, {value}, {name}, {index}, {output})
- Format icons based on workspace state (urgent, empty, focused, active)
- Clickable workspace navigation (can be disabled)
- Drag and drop workspace reordering
- Flexible output display (current output only or all outputs)
- Smart empty workspace handling
- Current-only mode for minimal display
- Window filtering via ignore rules
- Full CSS styling integration with state-based classes
- Real-time updates via Niri's event stream
